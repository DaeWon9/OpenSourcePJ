### 카카오프렌즈 컨셉으로 제작한 섯다 게임입니다.
#### 최고의 섯다꾼이 되고 싶은 라이언을 도와 각 레벨별 카카오프렌즈 친구들 상대로 승리하세요!

---------------------------------------

### 🎮 Game Description

#### 게임시작
- 시작 버튼을 클릭하여 게임 시작 가능
- 게임이 진행되는 도중에는 밖으로 나갈 수 없다 (뒤로가기 버튼 사용 불가)

#### 게임진행
- 사용자와 컴퓨터 중 차례가 “선”인 쪽부터 랜덤카드가 차례대로 분배된다.
- 사용자에겐 컴퓨터의 카드 2장 중 1장의 카드가 공개된다.
- 사용자는 본인에게 주어진 카드2장과 조합정보를 보여준다.
- 족보버튼을 사용하여 조합의 우선순위를 알 수 있다.
- 배팅은 한번 씩 차례대로 진행된다.
- 컴퓨터는 컴퓨터의 패에 맞게 오토배팅이 진행된다.
- 사용자는 아래쪽에 위치한 버튼으로 배팅을 진행한다.
- 처음 배팅 시에는 다이 & 하프만 가능하다. (컴퓨터도 동일)
- “다이”로 게임이 종료될 경우 죽은 쪽의 카드는 공개되지 않고 상대의 승리로 게임이 종료된다.
- “콜”로 게임이 종료될 경우 각자의 패가 공개되고 비교 후 승패가 결정된다.
- 승자가 다음게임 “선”을 가져간다.
- 보유금액이 0원 이하가 될 경우 보유금액에는 0원이 표시되고 사이드 배팅이 가능하다.
- 컴퓨터와 사용자 중 한명이 파산 할 경우 엔딩맵으로 이동한다.

#### 배팅
- “다이” : 게임을 포기한다.
- “콜” : 상대방이 배팅한 금액을 배팅하고 상대와 승부한다.
- “쿼터” : 배팅된 금액의 4분의1 만큼을 배팅한다.
- “하프” : 배팅된 금액의 2분의1 만큼을 배팅한다.
